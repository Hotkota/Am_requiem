# Am

[Discord server](https://discord.gg/rgsV2AQ)

## Installation

Install RedisDB and

```bash
git clone https://github.com/Hotkota/Am_requiem.git
cd Am_requiem
shards install
```

## Usage

```bash
crystal run src/am.cr
```
or

```bash
crystal build src/am.cr
./am
```
