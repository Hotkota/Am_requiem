# Am

[Discord server](https://discord.gg/rgsV2AQ)

## Installation

```bash
git clone https://github.com/Hotkota/Am_requiem.git
cd Am_requiem
shards install
```

## Usage

```bash
crystal run src/bot.cr
```
or

```bash
crystal build src/bot.cr
./bot
```
